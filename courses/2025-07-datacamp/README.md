## Course Philosophy: The Four P's Framework

**Plan** → **Prompt** → **Perfect** → **Publish**

Modern AI-assisted development that makes coding accessible, efficient, and secure for all skill levels. Focus on problems, not syntax.

## Voice Guidelines

**Embrace Matt's Natural Voice:**
- Direct observations: "Notice how..." "Watch what happens when..."
- Personal experiences: "I've seen..." "When I first..."  
- Conversational questions: "Ever felt like...?" "What if instead..."

**Avoid Generic Instructor-Speak:**
- ~~"Here's the thing about..."~~ → "Watch what happens..."
- ~~"At the end of the day..."~~ → "Bottom line:"
- ~~"It's important to note..."~~ → Just state it directly

## What Students Will Build

**Introduction Project:** Simple meditation timer (one-shot exercise to learn Replit basics)

**Main Project:** Complete form submission app with kanban dashboard and full GTM assets featuring:
- Authentication and user management
- Dynamic form builder with custom field types and validation
- Form submission processing and secure data storage
- Kanban dashboard with drag-and-drop response status management
- Response analytics and submission tracking
- Export functionality for response data (CSV download)
- Clean, modern interface
- Secure production deployment
- Complete GTM asset suite
