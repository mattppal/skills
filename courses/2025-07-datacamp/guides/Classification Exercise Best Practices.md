# Classification Exercise Best Practices

In classification exercises, learners classify items from a list into two or more groups by dragging cards from a stack and dropping them onto the correct target zone. You can [see an example of this exercise in action here](https://campus-no-cache.datacamp.com/courses/26093/84524?ex=2).

  

![](https://t24474497.p.clickup-attachments.com/t24474497/a34f22de-c638-44c4-8109-f73fc69df1cf/dnd_classify_gif.gif)

## The anatomy of a classification exercise

A classification exercise is comprised of different components:

  

*   Classification zones (the buckets to drag items into)
*   Items (the items that get assigned to different buckets)
*   An exercise title
*   An exercise description
*   Exercise instructions
*   Exercise hints

  

![](https://t24474497.p.clickup-attachments.com/t24474497/21b05fad-30af-4f65-a276-e2d2646c221e/image.png)

## Guidelines and best practices

*   There cannot be more than three classification zones
*   The maximum number of classification items that can be used in an exercise is 6.
*   The exercise title cannot exceed 25 characters.
*   The exercise description needs to embed an element of storytelling ([see here](http://The exercise description needs to embed an element of storytelling (see here)))
*   Instructions need to be clear and in bullet points (even if one)
*   Hints need to be clear, but not give away the answer, and in bullet points (even if one)
