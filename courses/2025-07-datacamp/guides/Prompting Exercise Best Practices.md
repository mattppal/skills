# Prompting Exercise Best Practices

Prompting exercises allow learners to interact with a chat-style interface where they input prompts to an AI model and observe its responses. These exercises are ideal for teaching prompt engineering, comparing prompt strategies, or analyzing how different phrasings influence outputs.

## Anatomy of a Prompting Exercise

Each prompting exercise consists of the following components:

  

*   **Chat interface** (side-by-side with the question)
*   **Exercise title** (max 25 characters)
*   **Exercise description** (includes narrative context; see [Storytelling Guidelines](http://#))
*   **Exercise instructions** (bulleted)
*   **Exercise hints** (bulleted and helpful but non-revealing)
*   **MCQ question** with 3–4 answer options (one or more correct, others plausible)

  
## Best Practices

Each prompting exercise consists of the following components:

*   **Align prompt goals with lesson objectives**: Make sure the outcome the learner is prompting for directly connects to the skill being taught (e.g., clarity, precision, tone control, syntactic correctness).
*   **Guide learners to think critically about the model's output:** Design MCQs that test prompt effectiveness.
*   **Include a visible grounding reference (optional):** If the exercise involves summarizing a dataset, code, or long text, include it visibly above or beside the prompt box for learner reference.
