# Multiple Choice Exercise Best Practices

The simplest kind of DataCamp exercise is a multiple-choice exercise. The learner chooses an answer to a question that doesn't involve any coding. There are two types of multiple-choice exercises:

  

*   Multiple Choice Single Answer: The learner can choose one answer option.
*   Multiple Choice Multiple Answers: The learner can choose various answer options.

## **Multiple Choice Exercises Have Several Parts**

1. This includes the background information that sets the stage and the question itself.
2. The possible answers.
3. The feedback messages correspond to each possible answer.
4. The hint that learners ask for when they are stuck.

  

The feedback messages and the hint are the keys to a great multiple-choice exercise. Telling the learner they're wrong without providing any guidance on what to do next is frustrating.

  

Telling them exactly which answer to pick instead can be equally frustrating; finding something in between takes practice, but is worth the effort.

## Guidelines

*   Exercise context is 30 - 550 characters (recommended is 500).
*   Possible answers are 10 - 300 characters for all possible answers (recommended is 240).
*   Between 3 - 5 possible answers (recommended is 4).
