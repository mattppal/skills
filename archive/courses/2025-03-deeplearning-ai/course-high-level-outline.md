# Vibe Coding 101: AI-Assisted Web Development

<introduction>
Educational course framework teaching accessible AI-assisted development. Aligned with Matt Palmer's "vibe coding" philosophy: making software creation intuitive and secure for all skill levels.
</introduction>

<mission>
Empower students to transform ideas into functional applications through evidence-based, hands-on learning with AI tools.
</mission>

## Course Overview

<context>
This course demonstrates how AI-assisted development democratizes software creation. Students learn to build complete web applications through practical projects, focusing on outcomes rather than syntax complexity.
</context>

## Learning Outcomes

<objectives>
By course completion, students will:

1. Transform ideas into working full-stack applications with persistent storage
2. Create structured development plans using AI-assisted planning
3. Design user-friendly interfaces with modern, accessible components
4. Build applications that collect, process, and visualize data effectively
5. Debug and troubleshoot applications using systematic approaches
6. Deploy and share creations with real users
7. Communicate effectively with AI tools to achieve development goals
</objectives>

## Prerequisites

- A willingness to learn
- Web browser (Chrome preferred)
- Replit account
- TLDraw account (free, optional)

## Projects

### Project 1: SEO Meta Tags Analyzer

An interactive tool that allows you to:

- Enter any website URL
- View a visual representation of the site's SEO meta tags
- Check if meta tags are properly implemented
- Get insights on SEO best practices

**Technical focus:**

- Web scraping and HTTP requests
- Data visualization
- Real-time validation

**Key skills:**

- Working with web APIs
- Processing and analyzing HTML content
- Creating intuitive data visualizations
- Error handling and edge cases

### Project 2: National Parks Voting App

A more complex web application that allows you to:

- Vote on National Parks in head-to-head comparisons
- See rankings calculated using a ranking system
- Create beautiful, engaging visuals
- View data imported from CSV files

**Technical focus:**

- Data management and storage
- Interactive UI components
- Algorithmic ranking systems
- Data visualization

**Key skills:**

- Database implementation
- Frontend development with modern frameworks
- Data processing and transformation
- User interaction design
