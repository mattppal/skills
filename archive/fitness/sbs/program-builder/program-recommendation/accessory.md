# Evidence-Based Accessory Programming Strategy

<primary_recommendation>
## Optimal Progression Schemes: Light Burgundy ("Rep Increase") or Gray ("Set increase then reps increase")

These progression schemes demonstrate superior alignment with current hypertrophy research through:

1. **Multi-variable progression**: Systematic overload through reps, sets, and load
2. **Research-validated rep ranges**: Maintains 6-20 rep hypertrophy zone
3. **Volume optimization**: Ensures adequate stimulus accumulation
4. **Measurable progression**: Clear performance metrics and advancement criteria
</primary_recommendation>

<implementation_protocol>
## Systematic Implementation Strategy

### For Isolation and Accessory Movements:
```
Baseline Parameters:
- Sets: 3-4 per exercise
- Reps: 8-12 initial range
- RIR: 1-3 on working sets, 0-1 on final set

Progression Model:
1. Increase reps across all sets to upper target
2. When upper rep range achieved, increase load 5-10%
3. Return to lower rep range with new load
4. Maintain consistent technique standards throughout
```

### Progressive Overload Sequence:
```
Week 1-3: 3 x 8-10 reps @RIR 2-3
Week 4-6: 3 x 10-12 reps @RIR 2-3
Week 7: Load increase, return to 3 x 8-10 @RIR 2-3
```
</implementation_protocol>

### Non-Recommended Schemes

**Teal (Classic Overload)**:
- Application: Main compound movements only
- Limitation: Excessive rigidity for accessory work
- Issue: Form degradation less manageable at higher intensities

**Yellow (Fixed Total Reps)**:
- Concern: Inconsistent per-set stimulus
- Problem: Difficult progressive overload tracking
- Result: Suboptimal adaptation signaling

**Peach (Set-by-Set)**:
- Complexity: Unnecessary for accessory movements
- Application: Reserved for primary lift variations
- Efficiency: Per-set load management not required

<research_foundation>
## Scientific Rationale

### Hypertrophy Research Summary:
- **Rep Range Effectiveness**: Meta-analyses demonstrate similar hypertrophy outcomes across 6-35 rep ranges when sets approach muscular failure
- **Volume Optimization**: 3-6 sets per exercise produce superior results compared to single-set protocols
- **Progressive Overload**: Multiple overload vectors (volume, intensity, frequency) drive continued adaptation
- **Proximity to Failure**: Regular exposure to 0-2 RIR enhances muscle growth stimulus

### Evidence-Based Parameters:
```
Volume Landmarks:
- MEV (Minimum Effective Volume): 4-6 sets per muscle group per week
- MAV (Maximum Adaptive Volume): 12-20 sets per muscle group per week
- MRV (Maximum Recoverable Volume): 20-25 sets per muscle group per week

Intensity Guidelines:
- Hypertrophy Focus: 65-85% 1RM
- RIR Targets: 0-3 for optimal stimulus
- Frequency: 2-3x per week per muscle group
```
</research_foundation>
