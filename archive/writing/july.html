🔍 Web Search for Agent

Agent can now search the web and access real-time information while building your projects. No more being stuck with outdated documentation or missing the latest framework updates.

Whether you need current API specifications, the newest library versions, or fresh examples from the community, Agent pulls in exactly what you need when you need it. Your builds stay current with the rapidly evolving development landscape.

This is the breakthrough that makes Agent truly powerful—unlimited access to the collective knowledge of the developer community, updated in real-time. Learn about Web Search.

🧠 Dynamic Intelligence

Agent just got a major brain upgrade. Dynamic Intelligence brings two game-changing capabilities that make building complex applications feel effortless.
  • Extended Thinking — gives Agent time to really analyze your requests before jumping in
  • High Power mode — upgrades you to Claude Opus for those challenging builds where precision matters most

Combined with Web Search, Dynamic Intelligence transforms Agent into a truly intelligent coding partner that thinks before it acts and has access to the latest information. This is the future of AI-assisted development. Learn about Dynamic Intelligence.

📝 Custom Agent Instructions (replit.md)

Your coding style matters, and now Agent learns exactly how you work. With replit.md files, you can teach Agent your preferences—coding standards, project structure, communication style, even specific libraries you love using.

Think of it as giving Agent a personal onboarding session for every project. Want Agent to use TypeScript by default? Prefer functional components in React? Have specific naming conventions? Just document it once in replit.md and Agent remembers.

This is personalization that actually makes you more productive. No more explaining the same preferences over and over—Agent just gets your style from day one.

💰 Effort-based Billing

We completely rethought how pricing works to make it fairer for everyone. Instead of fixed checkpoints, you now pay based on the actual work Agent does. Simple requests use fewer resources, complex builds use more—and you only pay for what you need.

Free users got the biggest upgrade: $3 worth of checkpoints instead of just 5 fixed ones. That means way more flexibility to build what matters to you, whether that's ten quick fixes or one substantial feature.

Building should be accessible to everyone, and this pricing model gets us closer to that reality. Learn about AI billing.

🌐 Domain Purchasing

Professional web presence just got easier. Domain purchasing is now available to all Core users—buy, configure, and manage domains directly in Replit without leaving your development environment.

Set up custom DNS records, move domains between apps, and give your projects the professional URLs they deserve. No more juggling external domain registrars or complicated DNS configurations.

Your ideas deserve to live at addresses that match their quality, and now that's as simple as clicking "Buy Domain" in your deployment settings. Learn about Domain Purchasing

🗂️ Agent Queue

Multi-tasking chaos is over. Agent now automatically queues your messages, handling one task at a time so nothing gets lost or confused. Send multiple requests and Agent works through them methodically.

No more worrying about interrupting Agent mid-task or losing track of what you asked for. Everything gets the attention it deserves, in the order you requested it. Learn about Message Queue.
